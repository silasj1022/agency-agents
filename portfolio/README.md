# Justin Silas - Public AI Engineering Portfolio

This directory contains original, public-facing work by Justin Silas. It is separate from the upstream agent-library content elsewhere in this repository.

## Featured project

### [Forward-Deployed AI Delivery Lab](forward-deployed-ai-lab/)

A runnable synthetic-data implementation demonstrating:

- Retrieval-grounded responses with source citations
- Policy guardrails and safe redirection
- Human-review routing for consequential actions
- Golden-set evaluation and release-gate metrics
- Stage-level telemetry and append-only audit events
- A browser demo and Replit-ready configuration

The project distills delivery patterns from larger private implementations while excluding proprietary code, credentials, employer data, student data, and internal security findings.

## Role alignment

The portfolio is designed for Lead/Principal forward-deployed AI engineering opportunities requiring customer discovery, rapid prototyping, enterprise data integration, agentic workflows, evaluation, security, governance, and production-readiness leadership.
